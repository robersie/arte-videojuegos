Nosotras

¿Por qué hemos elegido este tema?

Hemos elegido este tema porque al gustarnos los videojuegos y el arte, nos hemos puesto de acuerdo para crear esta página web para que otras personas puedan aprender otras cosas diferentes.

Nuestra experiencía en Code Like A Girl

Hemos aprendido a programar gracias a un curso de vodafone llamado #CodeLikeaGir. Este curso se ha hecho para que todas las chicas sepan que pueden programar o dedicarse a la robótica igual que los hombres.
